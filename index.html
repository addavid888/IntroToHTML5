Introduction to HTML5

HTML5 is a markup language used for structuring and presenting content on the World Wide Web.

Why HTML is called a markup language?

  • Markup symbols: These symbols, or codes, are inserted into a file that will be displayed on the internet.
  • Structure: The markup describes how the content in the file is structured.
  • Display: The markup tells the web browser how to display the content, such as words, images, and other multimedia.
